# Product images

Drop product photos here (e.g. `banana-slicer.png`) and reference them in
`data/videos/*.json` as `/products/banana-slicer.png`.

The sample data in `data/videos/1001.json` and `1002.json` references image
files that don't exist yet (`banana-slicer.png`, `vegetable-chopper.png`,
`jar-opener.png`, `usb-charger.png`) — add real photos with these exact
names, or edit the JSON to point at whatever files you add. Recommended:
square images, at least 500x500px, optimized/compressed (tinypng.com) before
committing so the site stays fast.
